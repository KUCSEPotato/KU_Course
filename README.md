## 소개

이 저장소는 고려대학교 컴퓨터학과에서 수강한 과목들의 과제, 프로젝트, 실습 자료를 정리한 개인 학습 아카이브입니다.

## 학기별 과목

### 2023학년도

#### 2023-1학기
- **SOCI114**: 정보사회론

#### 2023-2학기
- **COSE221**: 논리설계
  - 4-bit Full Adder, HEX ADDER 등 Verilog 설계 과제
- **GEST137**: 생물다양성과진화

---

### 2024학년도

#### 2024-1학기
- **COSE213**: 자료구조
  - 과제1-4: 기본 자료구조 및 알고리즘
  - 과제5: Double Linked List ADT
  - 과제6: Binary Search Tree 구현
  - 과제7: Heap ADT 구현
  - 과제8: AVL Tree 구현
  - 과제9: Permuterm Trie 구조

- **COSE215**: 계산이론

#### 2024-2학기
- **COSE222**: 컴퓨터구조
  - RISC-V Single Cycle 구현
  - RISC-V Multi Cycle 구현

- **COSE342**: 컴퓨터네트워크
- **COSE371**: 데이터베이스
- **COSE382**: 확률및랜덤과정

---

### 2025학년도

#### 2025-1학기
- **COSE341**: 운영체제
  - Assignment_OS_2025: 운영체제 과제

- **COSE451**: 소프트웨어보안
  - Assignment 0-3: 4개의 과제

#### 2025-2학기
- **COSE354**: 정보보호
  - Assignment1: CMINE 관련
  - Assignment2: Secure Coding and Design
  - CTF 과제

- **COSE474**: 딥러닝
  - Assignment 1-2
  - Term Project

## 주요 프로젝트 및 과제

### 하드웨어 설계 (COSE221)
- Verilog를 이용한 디지털 회로 설계
- 4-bit Full Adder, HEX Adder 구현

### 자료구조 (COSE213)
- C 언어를 이용한 다양한 자료구조 구현
- Linked List, BST, AVL Tree, Heap, Trie 등

### 컴퓨터구조 (COSE222)
- RISC-V 프로세서 설계
- Single Cycle 및 Multi Cycle 구현

### 정보보호 (COSE354)
- 시큐어 코딩 및 설계
- CTF 챌린지

### 소프트웨어보안 (COSE451)
- CTF 챌린지

### 딥러닝 (COSE474)
- 딥러닝 알고리즘 구현
- 텀 프로젝트
