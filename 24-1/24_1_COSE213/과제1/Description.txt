# 동적 배열 구현

주어진 소스코드를 완성한다

- 소스파일에는 함수의 원형이 선언되어 있으며, 함수를 직접 정의해야 함.
- main 함수에 있는 주석을 제거하여 실행 파일과 같이 실행되도록 수정해야 함
- valgrind를 통해 메모리 누수 현상을 제거해야 함
- 과제 설명에 대한 강의는 “강의자료 → 3주차”에 있음
- 실행방법
    
    $ ./word_count1 -n words.s.txt > 출력파일명
    
    $ ./word_count1 -f words.s.txt > 출력파일명
    
- 제출 파일: word_count1.c (파일명을 수정하거나 압축하지 말 것!)
- 제출 마감 4/3 (수) 23:59:00 까지

실행할때 valgrind 체크하기

$valgrind ./a.out -n words.s.txt > result.n 
$valgrind ./a.out -f words.s.txt > result.n

작성해야하는 함수는 원형이 선언되어 있는 함수들.

word_count:
한줄 한줄 읽어가면서 사전에 추가
사전에 없는 것은 사전에 추가하고 있는 것은 빈도만 1씩 올리기
이번 과제는 바로 앞에 읽었던 단어와 지금 읽고 있는 단어만 확인해보면 새 단어인지 확인 가능
→ 이것이 힌트임
→ 단어 크기 터질때 재할당 시키는 것도 포함해야함

print_dic → 사전을 화면에 출력
destroy_dic → 할당 받은 메모리 반환 // 문자열들 해제, 단어 해체, 사전 해체
compare_by_word → 단어 기준 정렬
compare_by_freq → 빈도 내림차순 기준 단어 정렬
